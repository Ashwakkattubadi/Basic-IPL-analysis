    "Finds which team won the most matches in a specific IPL season.\n",
    "\n",

    "\n",
    "Shows the IPL champion for a selected season.\n",
    "\n",
    "### 4. Match Winner Details\n",

    "* Winning margin\n",
    "* Toss winner\n",
    "* Toss decision\n",

    "* Match venue\n",
